## 🚀 Overview

Clevision is a conceptual scalable web application layout that emphasizes superior UI/UX, seamless responsiveness, and interactive state management. It was developed to showcase modern frontend design paradigms, including abstract micro-gradient backgrounds, sticky glassmorphism navigation, and dynamic content rendering.

## ✨ Key Features

- **📱 Mobile-First Responsive Design:** Flawlessly adapts across all device breakpoints (Mobile, Tablet, Desktop).
- **🎨 Premium Dark Mode Theme:** Custom Tailwind configuration utilizing a sleek Midnight Zinc and Vibrant Violet color palette.
- **🛒 Interactive Product Catalog:** Dynamic grid layout featuring live state management for filtering and sorting (Price: Low to High / High to Low).
- **📊 Embedded Analytics Engine:** Real-time array reduction and object mapping for in-stock inventory values (logs directly to the browser console).
- **🍔 Custom Mobile Navigation:** Smooth slide-in hamburger menu with backdrop blurring using Lucide React icons.

## 🛠️ Tech Stack

- **Core:** [React 18](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/) for lightning-fast HMR and optimized builds.
- **Styling:** [Tailwind CSS v3](https://tailwindcss.com/) for utility-first styling and custom UI components.
- **Icons:** [Lucide React](https://lucide.dev/) for clean, consistent SVG iconography.
